# 🛍️ Perfumaria E-commerce

Este é um projeto de uma página inicial para uma **Loja Virtual de Perfumes e Cosméticos** totalmente responsiva. A interface foi construída utilizando **HTML5** estruturado combinando o poder do **Bootstrap 5** com estilos customizados em **CSS3** para garantir uma experiência de navegação fluida em computadores, tablets e smartphones.

---

## 🚀 Funcionalidades da Interface

* **Barra de Navegação Dinâmica:** Menu responsivo otimizado para navegação entre categorias principais (Perfumes, Corpo e Banho, Linha Ambiente e Promoções)[cite: 46].
* **Banner Rotativo (Carousel):** Destaque de campanhas publicitárias na página principal com transição automática de slides[cite: 46].
* **Mural de Benefícios:** Seção visual com ícones circulares destacando vantagens competitivas como Frete Grátis, Parcelamento e Certificados de Segurança (SSL)[cite: 46].
* **Vitrine de Produtos:** Grid responsivo exibindo os "Últimos Lançamentos" com cards limpos, preços destacados e opções de parcelamento[cite: 46].
* **Rodapé Informativo (Footer):** Divisão clara em colunas estruturando informações institucionais, institutos de suporte e categorias do site[cite: 46].

---

## 🛠️ Tecnologias e Recursos Utilizados

* **HTML5:** Estrutura semântica das seções (Navbar, Carousel, Grid de Produtos, Footer)[cite: 46].
* **Bootstrap 5 (v5.3.8):** Framework utilizado para toda a base do layout, sistema de grids responsivos (`row`, `col`), alinhamento flexível e componentes interativos[cite: 46].
* **Bootstrap Icons (v1.13.1):** Biblioteca de ícones vetoriais integrada para ilustrar os diferenciais e facilidades de compra[cite: 46].
* **CSS3 Customizado:** Estilização aplicada internamente para ajustes finos de identidade visual, como a criação do fundo neutro (`.bg-gray`)[cite: 46].
* **Git Configuration:** Normalização automática de quebras de linha (`.gitattributes`) definida para o padrão de repositório[cite: 45].
